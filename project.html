Project: bubbl

Description

bubbl is a mobile web application that helps people form new friend groups by matching them with local events based on mutual interest. Rather than swiping on individuals, users swipe on nearby events—sourced from external APIs like Google Maps or Eventbrite—and are matched into spontaneous social groups ("bubbles") with others who swiped right on the same event. Once matched, participants are given a day or two to chat, plan, and then meet up in person to participate in the event.

Usefulness

Many people want to meet new friends but find it difficult to do so organically. While dating apps like Tinder and Bumble BFF exist, and event apps like Meetup or Facebook Events help people discover activities, few platforms actively match users into small groups around mutual interest in local events. bubbl fills this gap by combining interest-based event discovery with social matching. Unlike Meetup, which requires long-term commitment or heavy planning, bubbl encourages spontaneity and lightweight social engagement.

Realness

We use real-world event data pulled from the Google Places API or Eventbrite, filtered by location and activity type. A sample of JSON-formatted event data has already been collected for the South Bend area, including event titles, categories, descriptions, locations, and timestamps. This will form the basis for our prototype and testing.

Functionality

Basic Functions

  • User account creation and login
  • Event browsing with swipe left/right interface
  • Formation of match groups based on mutual right swipes
  • In-app messaging within groups
  • Viewing upcoming matched events

Advanced Functions

  • Dynamic Group Formation: Real-time matchmaking system that forms temporary chat groups around events as soon as a threshold of right swipes is reached. Includes expiration logic to disband inactive groups.
  • Interest & Personality Tag Matching: Users can optionally add interests and personality tags, which are used to prioritize group formation to ensure compatibility beyond event preference. This will require implementing tag-based similarity scoring and weighted matchmaking algorithms.

Why It's Cool

bubbl makes it easy to try something new with new people. It eliminates the pressure of 1:1 meetups or ongoing commitments, and encourages users to step outside their comfort zones. The ephemeral nature of the groups keeps things low-stakes while still encouraging real-world interaction. Features like interest-based matching and real-time group thresholds add a layer of technical depth and user personalization not commonly found in event discovery apps.
